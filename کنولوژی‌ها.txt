تکنولوژی‌های قطعی (The Tech Stack)
اینها ابزارهایی هستند که در پروژه استفاده می‌کنیم. انتخاب‌ها بر اساس مهارت‌های فعلی تو (پرامیت ۵)، نیاز پروژه، و استانداردهای صنعتی انجام شده است:

حوزه	تکنولوژی انتخاب‌شده	دلیل انتخاب
Frontend Core	React (با Vite)	مهارت اصلی تو. Vite سرعت توسعه را بالا می‌برد.
Routing	React Router DOM v6	نیاز به صفحات تو در تو (Nested)، محافظت‌شده (Protected)، و مدیریت URL داریم.
Server State	TanStack React Query v5	برای Fetch، Cache، و همگام‌سازی خودکار داده‌ها با سرور. یادگیری آن در حین پروژه.
Backend & Auth	Supabase	دیتابیس PostgreSQL، احراز هویت، و RLS (امنیت سمت سرور) را یکجا فراهم می‌کند.
Client State	Context API + useReducer	برای وضعیت‌های سراسری پیچیده مثل Auth و Theme. معماری "گوینده و بلندگو" خودت.
Styling	CSS Modules	مهارت تثبیت‌شدهٔ تو. ماژولار، بدون تداخل استایل، و نیاز به یادگیری ابزار جدید ندارد.
Notifications	React Hot Toast	برای بازخورد لحظه‌ای (مثل "پروژه با موفقیت ساخته شد") به کاربر.
Icons	React Icons	کتابخانهٔ سبک و کامل برای آیکون‌های رابط کاربری.
